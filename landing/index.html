Librarian AI

Read academic PDFs in an extension-controlled viewer: highlights, notes, and chat with context from your selection—without leaving the page.

View source Chrome Web Store Latest release

What it does

Owned viewer

PDFs open in a first-party PDF.js surface so highlights and the sidebar stay stable.

Highlights & notes

Capture quotes, jump back to the source, and attach notes per highlight.

AI chat

Ask questions with thread context from what you selected; API key stays local in the extension.

Install from source

  1. Clone the repo from View source above.
  2. In Chrome, open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and choose the Librarian AI folder (the one that contains manifest.json).
